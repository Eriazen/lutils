  • str_id (str) – The string pattern or regex to search for.

  • return_next (bool, optional) – If True, returns the non-whitespace string immediately following the match. If False, returns True immediately upon finding a match. Default is True.

Returns:

str or bool or None

lutils.utils.get_of_version(case_path: Path) → int | None[source]¶

Extracts the OpenFOAM version number from the case’s solver log.

This function first identifies the solver used via ‘system/controlDict’, then searches the corresponding log file for the version string.

Parameters:

case_path (Path) – The root directory of the OpenFOAM case.

Returns:

The major version number if found (e.g., 2312, 11), otherwise None.

Return type:

int or None

Raises:

ValueError – If ‘system/controlDict’ is missing or if the solver name cannot be found within it.
